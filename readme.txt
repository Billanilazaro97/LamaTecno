🖥️ Proyecto de LamaTecno
Este proyecto es una página web simple y elegante que presenta una serie de imágenes de productos, enlaces de navegación y una sección de contactos.


🌐 Contenido de index.html
El archivo index.html contiene la siguiente estructura:

Banner: Un div vacío que puede ser utilizado para un banner.
Header: Contiene una barra de navegación con enlaces a "Inicio", "Productos" y "Contactos".
Contenedor: Incluye una serie de imágenes de productos.
Footer: Contiene información de contacto y enlaces a redes sociales.


📄 Descripción de la Página
🧭 Barra de Navegación
Permite la navegación entre las diferentes secciones de la página.

Inicio: Enlace a la página principal.
Productos: Enlace a una página con más detalles sobre los productos (Productos.html).
Contactos: Enlace a la sección de contactos al final de la página.


📞 Footer
Información de contacto con enlaces a WhatsApp, Facebook e Instagram.

Contacto: Sección de contactos con el título "Contactos" y un subtítulo sobre envíos a domicilio.
